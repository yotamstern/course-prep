git bisect: a command to help find bugs from previous commits using a binary search. You provide the command an example of a good commit and a bad one and it and the command uses that to find the first commit that introduced the bug. Since it uses binary search it'll run in log(n) time. This command is powerful because it makes finding a bad commit easier and faster and automates the process.
