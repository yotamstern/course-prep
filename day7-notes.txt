FHS- 

.git/config - contains the basic git configs for my repo and it defines what is the url of the remote repo. it also stores the tracking relationship between your local branch and the remote branch 

.git/refs/remotes/ -  this directory contains your remote-tracking refs — one file per tracked remote branch, and each file contains a single commit hash. That hash is where the remote branch was pointing as of your last fetch. It's not the remote's objects — the objects themselves live in .git/objects/ regardless of whether they came from local work or a fetch.
