/ - root of all of the filesystem

/bin - contains exe files in single user mode

/boot - contains configs for the bootloader

/dev - contains the devices connected to the machine

/etc - config files that are local to the machine

/home - home directory for users on the machines

/lib - contains libraries for the system and commands in the root filesystem
 
/media - mount points for removable media like CDs and DVDs

/proc - start of proc filesystem (info on running processes and kernel)

/root - home dir for root user 

/run - info on the system since it started the boot up

/sbin - commands to boot the system (like default kernel)

/tmp - temporary filesystem

/usr - start of user specific filesystem

/usr/bin - primary dir for exe programs. 

/usr/local - local programs

/var/log - log files 

/usr/local/bin - binaries for local programs 

/var - contains variable files that can change

/mnt-  mount point for tmp mounted filesystems

/opt - contains add on packages to the distro

/sys - mount point for sysfs filesystem that contains info on kernel
